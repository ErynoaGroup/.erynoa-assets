# .erynoa-assets — Bootstrap

> Statische Assets für die Erynoa Group.

## What

Zentrales Repo für Icons, Logos, Bilder, Fonts. Public, damit Assets direkt per Raw-URL eingebunden werden können.

## Code

- `icons/` — App Icons, Favicons
- `logos/` — Brand Logos (SVG, PNG)
- `images/` — Allgemeine Bilder
- `fonts/` — Web Fonts
