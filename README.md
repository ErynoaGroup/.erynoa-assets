# Erynoa Assets

Static assets for the Erynoa Group — icons, logos, images, fonts.

## Structure

```
├── icons/       # App icons, favicons
├── logos/       # Brand logos (SVG, PNG)
├── images/      # General images
└── fonts/       # Web fonts
```

## Usage

Raw URL pattern:
```
https://raw.githubusercontent.com/ErynoaGroup/.erynoa-assets/main/{path}
```
